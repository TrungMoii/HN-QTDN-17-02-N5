## 🐛 TROUBLESHOOTING

### UI không hiển thị đúng?
```
1. Clear browser cache (Ctrl+Shift+Del)
2. Hard reload (Ctrl+F5)
3. Check Console (F12) → Any CSS errors?
4. Verify CSS files loaded (Network tab)
```

### Animations không mượt?
```
1. Check GPU acceleration enabled
2. Close unnecessary browser tabs
3. Update graphics drivers
4. Try Chrome (best performance)
```

### Colors không đúng?
```
1. Verify module upgraded (not just installed)
2. Check CSS file path in manifest
3. Restart Odoo server
4. Re-upgrade module
```

### Mobile responsive issue?
```
1. Test with real device (not just resize)
2. Check viewport meta tag
3. Test in different browsers
4. Report specific screen size issue
```
